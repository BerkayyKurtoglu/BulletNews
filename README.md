### This app uses :point_down:
- MVVM (model-view-view model) software structure
- Working on Compose migration
- **Paging 3** Library
- Hilt Dependecy Injection
- Retrofit
- RxJava2
- Kotlin Coroutines - job
- Data Binding
- View Binding
- Room Database
- Jetpack Navigation Component
- Disposable
- Glide
